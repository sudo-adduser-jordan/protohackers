# java protohackers

java solutions and tests for protohackers

# install deps

```sh
curl -s "https://get.sdkman.io" | bash
source "/root/.sdkman/bin/sdkman-init.sh"
sdk install java 21.0.2-graalce
sdk install gradle 9.0.0
```

# run server

```sh
gradle d0
```

```sh
gradle d1
```

```sh
gradle d2
```
